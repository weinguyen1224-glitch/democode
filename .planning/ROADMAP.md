# ROADMAP

## Phase 1 — Setup and Base Styles
**Goal:** Initialize project structure, base CSS variables, typography, and Shorthand integration
**Success Criteria:**
- CSS variables match UI-SPEC color palette
- Typography scales responsive
- No console errors on load

## Phase 2 — Chapter I Nguồn gốc
**Goal:** Implement scroll reveal animations for 5 sections (Vùng đất → Chiếc bánh)
**Success Criteria:**
- 5 sections with scroll-triggered reveals
- Parallax background on first section
- Zoom transition to Chapter II

## Phase 3 — Chapter II Ngũ Hành Interactive
**Goal:** Build interactive Ngũ Hành component with 5 elements orbiting center
**Success Criteria:**
- Click to highlight + show meaning
- Mobile fallback as vertical list
- CSS-first animations

## Phase 4 — Chapter III Kỹ nghệ Infographic
**Goal:** Circular infographic with 6-step process flow
**Success Criteria:**
- Circular layout on desktop
- Vertical timeline on mobile
- Scroll/click to reveal steps

## Phase 5 — Chapter IV Before/After Slider
**Goal:** Traditional vs Modern comparison slider
**Success Criteria:**
- Draggable handle (desktop)
- Tab toggle (mobile)
- Smooth transitions

## Phase 6 — Chapter V Audio and Frame Animation
**Goal:** Audio player + 4-frame scroll animation + closing scene
**Success Criteria:**
- Custom audio UI with transcript
- Frame animation tied to scroll
- Reduced motion support

## Phase 7 — Polish and QA
**Goal:** Final responsive polish, accessibility, performance
**Success Criteria:**
- All breakpoints tested (375px, 768px, 1440px)
- prefers-reduced-motion handled
- Zero console errors
